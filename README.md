# Назначение

Преобразование файла Государственный сводный реестр лицензий из XML-файла в CSV-формат - https://fsrar.gov.ru/opendata/7710747640-reestr

Преобразование XML-файла в CSV-формат с возможностью выбора определенных столбцов. Представлено готовое программное обеспечение в виде исполняемого файла (.exe), которое включает конфигурационный файл и поддерживает передачу параметров через командную строку.

# Файл параметров config.json

Файл конфигурации **config.json** генерируется при первом запуске системы и является обязательным для заполнения.

| Параметр          |                                                                                                                                                                                                                                          Назначение |
|:------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------:|
| as_fields         |                                                                                                                                                          Перечень полей XML к выгузке через запятую<br/>Если не указано хотя бы одно поле остановка |
| as_isdebug        |                                                                                                                                                                                         Отладка - вывод в консоль<br/> true - остановка по presskey |
| as_max_row_count  |                                                                                                                                                                        Отладка - получение ограниченого кол-ва первых строк</br>0 - без ограничений |     
| as_is_use_filter  |                                                 Фильтровать строки по полю 'Дата_окончания_действия_лицензии'<br/>Сохраняются только строки с значением >= настройки<br/>Ожидается строка (в формате DD.MM.YYYY), пустое значение - без ограничений |
| as_is_only_unique | Только уникальные записи<br/>Номер_лицензии__соответствующий_номеру_записи_в_реестре<br/>ИНН_организации_сельскохозяйственного_товаропроизводителя_<br/>КПП_обособленного_подразделения_организации__осуществляющего_лицензируемый_вид_деятельности | 
Например:

```json
{
  "as_fields": "Полное_и_сокращенное_наименование_организации_сельскохозяйственного_товаропроизводителя_с_указанием_ее_ОПФ,ИНН_организации_сельскохозяйственного_товаропроизводителя_",
  "as_isdebug": false,
  "as_max_row_count": 0,
  "as_is_use_filter": "01.01.2026",
  "as_is_only_unique": true
}
```

# Запуск из командной строки

При указании параметров следует использовать двойные кавычки. Рекомендуется избегать применения DNS-имен и наименований, содержащих пробелы или символы, не относящиеся к латинскому алфавиту.

Например:
```
fsrarXMLtoCVS.exe "c:\...\XML\data-20260701t0000-structure-20190918t0000.xml" "192.168.1.200\\...\CSV\result.csv"
```

| Параметр |                                                                                                                                       Назначение |
|:---------|-------------------------------------------------------------------------------------------------------------------------------------------------:|
| 1        |                                                                                     Полный путь к файлу XML <br/>- Если файл не найден остановка |
| 2        | Полный путь к создаваемому файлу CSV<br/>- Если каталог не обнаружен остановка<br/>- При повторном запуске результирующий файл будет перезаписан |
